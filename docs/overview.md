## Getting Started

In this section, we’ll prepare the environment needed to complete the lab. Please follow each step carefully to ensure everything is ready for the hands-on exercises. Once the environment is configured, you’ll be ready to dive into the lab activities!

To get started, you’ll need to download the Postman collection for this lab. This collection contains all the necessary API requests to complete the exercises. Follow these steps:

1. Download the Collection: <a href="https://github.com/WebexCC-SA/LAB-2851/blob/main/docs/assets/Wx1_Simplified_JDS_Collection.postman_collection.json" target="_blank">WebexOne JDS Postman Collection</a>  - Click on the link to download the Postman collection file.
2. Import into Postman:
    - Open Postman.
    - Go to File > Import.
    - Select the downloaded collection file and import it.

Once imported, you’ll be able to access the collection and use it to complete the lab tasks.
